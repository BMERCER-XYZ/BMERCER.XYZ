Mobile Vehicle Maintenance Scheduler
$ version 1.0.0 | status: ACTIVE DEVELOPMENT_
Project Overview
Platform
IOS application
Primary Function
Vehicle maintenance scheduling and reminders
Development Stack
SwiftUI | IOS | CoreData

Maintenance Tracker is an IOS application designed to manage vehicle maintenance schedules with reminders and logging. It was designed with the goal of making vehicle maintenance as simple as possible

Development Roadmap

Performance

Privacy Policy

Privacy and data security were carefully considered in the creation of this appplication.

Data Collection

Logging vehicle details and maintenance times from the watchOS app is accomplished using the WatchConnectivity framework. As with all Apple frameworks, it's incredibly robust and secure. For more information on exactly how it works please follow this link: WatchConnectivity Documentation.

Data Storage & Security

User data generated or observed within Maintenance Tracker is securely stored either on device or within iCloud. Data sharing/ access between the IOS and WatchOS app is accomplished using App Groups. For more information on how App Groups work please follow this link: App Groups Documentation. I can assure you that I have zero interest in any of your data.

Third-Party Sharing

Motorcycle Maintenance Tracker does not sell, store, trade, or share personal data with third parties.
