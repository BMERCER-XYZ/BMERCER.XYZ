Mobile Motorcycle Maintenance Tracker & Scheduler
$ version 1.0.0 | status: ACTIVE DEVELOPMENT_
Project Overview
Platform
IOS application
Primary Function
Vehicle maintenance scheduling and reminders
Development Stack
SwiftUI | IOS | CoreData

MotoTrack is an IOS application designed to manage vehicle maintenance schedules with reminders and logging. It was designed with the goal of making motorcycle maintenance as simple as possible.

Development Roadmap
COMPLETED
Core Infrastructure
Basic UI framework. New bike sheet, settings sheet, add maintenance sheet etc.
COMPLETED
Data Import/Export
Data import/export support for bike details and maintenance logs via CSV files
PLANNED
Reminders/ Notifications
Push notifications/ reminders for upcoming maintenance tasks
PLANNED
Performance
Performance optimization for both IOS and WatchOS
Privacy Policy

Privacy and data security were carefully considered in the creation of this appplication.

Data Collection & Storage

User data generated or observed within MotoTrack is securely stored on device using coredata. I can assure you that I have zero interest in any of your data.

Third-Party Sharing

MotoTrack does not sell, store, trade, or share personal data with third parties.
