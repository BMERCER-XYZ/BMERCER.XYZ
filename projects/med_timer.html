Primary Function

Development Stack

Med Timer is an IOS & WatchOS application designed to manage medication schedules with reminders and logging. It was designed with the goal of making medication management as simple as possible

Privacy Policy

Privacy and data security were carefully considered in the creation of Med Timer.

Data Collection

Logging dose times from the watchOS app is accomplished using the WatchConnectivity framework. As with all Apple frameworks, it's incredibly robust and secure. For more information on exactly how it works please follow this link: WatchConnectivity Documentation.

Data Storage & Security

User data generated or observed within Med Timer is securely stored either on device or within iCloud. Data sharing/ access between the IOS and WatchOS app is accomplished using App Groups. For more information on how App Groups work please follow this link: App Groups Documentation. I can assure you that I have zero interest in any of your data.

Third-Party Sharing

Med Timer does not sell, store, trade, or share personal data with third parties.
