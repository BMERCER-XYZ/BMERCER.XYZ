Custom Fabricated CNC Drawer
$ version 1.0.0 | status: COMPLETED_
Project Overview
Application
Storage for various tools and materials for CNC machining
Development Stack
CAD | Welding

A custom fabricated CNC drawer designed for optimal storage and organization of tools and materials. The project utilizes CAD for design and welding techniques for construction, ensuring durability and functionality.

  • •
    Sliders

Features

  • •
